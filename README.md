# AI-Exercise-Recommender
An AI Exercise Recommender is a web application that suggests exercises based on the body part selected by the user. The system highlights the targeted muscles on a human body diagram and displays exercises that train that area.
Features
1] Body Part Selection
Clickable human body diagram
Dropdown search for body parts
2] Exercise Recommendations
List of exercises for selected muscle
3] Muscle Highlight
Shows targeted muscles on a body diagram
4] Exercise Details
Difficulty level
Equipment needed
Muscle group
5] Exercise Animation
GIF or video demonstrating exercise
6] Voice Search (Optional)
User speaks exercise name
7] Workout Builder (Advanced)
Create a full workout plan
